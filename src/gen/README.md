# gen

Files in this folder are automatically generated. DO NOT EDIT!

## base
Files in folder `base` are generated using the `ghdl_synth.sh` script.
To regenerate run:
```sh
cd build
./ghdl_analyze.sh
./ghdl_synth.sh
```

## opt
Files in folder `opt` are generated using the `yosys_synth.sh` script.
To regenerate run:
```sh
cd build
./ghdl_analyze.sh
./yosys_synth.sh
```

## altera
Files in folder `altera` are generated by Megawizard GUI in Quartus and are based on the `fp_functions` IP block.

The general configuration is:
 - Format = Custom
 - Exponent = 5 bits
 - Mantissa = 10 bits
 - Rounding = relaxed
 - Goal = Frequency
 - Target = 200 MHz
